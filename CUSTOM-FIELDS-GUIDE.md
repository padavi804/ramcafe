# Custom Fields Guide - Editing About Page Content

The About page template now has a **beautiful custom editing interface** built right into WordPress! No plugins needed, no complicated field names to remember.

## What Changed?

The layout and design stay the same (beautiful!), but now you get a user-friendly form to edit all the content directly in the WordPress page editor.

---

## How to Edit Content in WordPress

### Step 1: Open the About Page

1. Log into your WordPress admin
2. Navigate to **Pages > All Pages**
3. Click to edit the **About** page
4. Make sure the template is set to **About/Mission Page** (in the right sidebar under "Page Attributes")

### Step 2: Use the Content Settings Box

You'll see a new box called **"About Page Content Settings"** right below the main content editor. This box contains organized fields for editing all sections:

---

## What You Can Edit

### 1. Mission Statement (Main Content Editor)
Your mission statement goes in the **main content editor** at the top of the page (above the Content Settings box). Type it there like any normal page content.

**Example:**
```
To provide a welcoming space where individuals with memory loss and their
care partners can connect, share experiences, and find support in a
judgment-free community.
```

---

### 2. Memory Café Section

In the **"About Page Content Settings"** box, you'll find:

#### Section Heading
- **Field label**: "Section Heading"
- **What it controls**: The "What is a Memory Café?" heading
- **Default**: Shows "What is a Memory Café?" if left blank
- **Example**: Change to "About Our Memory Café" or "Memory Café Overview"

#### Section Description
- **Field label**: "Section Description"
- **What it controls**: The paragraph explaining what Memory Cafés are
- **Default**: Shows pre-written description if left blank
- **Tip**: Write 2-3 sentences explaining what makes your Memory Café special

---

### 3. Feature Boxes (3 boxes total)

Each feature box has its own group with clear labels:

#### Feature Box 1 - Social Connection
- **Title field**: Change "Social Connection" to your preferred title
- **Description field**: Edit the supporting text
- **Icon**: Users/People icon (not editable)

#### Feature Box 2 - Safe & Welcoming
- **Title field**: Change "Safe & Welcoming" to your preferred title
- **Description field**: Edit the supporting text
- **Icon**: Smiley face icon (not editable)

#### Feature Box 3 - Engaging Activities
- **Title field**: Change "Engaging Activities" to your preferred title
- **Description field**: Edit the supporting text
- **Icon**: Layers icon (not editable)

---

## Quick Example: Editing Content

Let's say you want to customize the first feature box:

1. **Open the About page** in WordPress editor
2. **Scroll down** to the "About Page Content Settings" box
3. Find **"Feature Box 1 (Social Connection)"**
4. In the **Title field**, type: `Build Connections`
5. In the **Description field**, type: `Join a caring community that understands your journey and provides meaningful support.`
6. Click **Update** (blue button in top right)
7. View your page - changes appear immediately!

---

## Tips & Tricks

### Using Default Text
- **Leave any field blank** to automatically use the default text
- This makes it super easy to reset - just clear the field and save

### Previewing Changes
- Click **Preview** button (next to Update) to see changes before publishing
- Changes only appear on the live site after you click **Update**

### Best Practices
- Keep titles short and punchy (2-4 words)
- Keep descriptions to 1-2 sentences for best layout
- Use clear, welcoming language that matches your mission

### What You Can't Change
- Box icons (requires code editing)
- Colors and styling (requires CSS changes)
- Number of boxes (currently fixed at 3)
- Layout and positioning (requires template changes)

## Troubleshooting

### "I don't see the About Page Content Settings box"
- Make sure you're editing a page (not a post)
- Check that the template is set to "About/Mission Page"
- Try refreshing the page or logging out and back in

### "My changes aren't showing on the front page"
- Make sure you clicked **Update** (not just Preview)
- Hard refresh your browser: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check that you're viewing the correct page

### "I want to reset to defaults"
- Simply clear all the fields (make them blank)
- Click **Update**
- The default text will automatically appear
