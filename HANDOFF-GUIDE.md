## Common Questions to Anticipate

**"What if I break something?"**
- We have backups! Easy to restore
- Most changes can be undone
- I'm available if you need help
- WordPress is designed to prevent major breaks

**"How often do I need to update it?"**
- Content: As often as you want (weekly, monthly)
- WordPress/Plugins: Check monthly, update quarterly
- We set up automatic backups

**"What if I forget how to do something?"**
- Check the README.md guide
- Google the task (WordPress has huge community)
- Email me (if we have support arrangement)
- Call SiteGround support (they can help with basics)

**"Can we change the design later?"**
- Colors: Yes, easy in Customizer
- Layout: Need developer (or child theme)
- Logo: Yes, easy to change
- Content: Yes, change anytime

**"How much will this cost ongoing?"**
- Domain: $13/year
- Hosting: $96/year (after first year)
- Plugins: Free (the ones we recommend)
- Updates/Maintenance: Your time or hire help
- Total: ~$110/year minimum

**"Do we own this?"**
- Yes! Domain is in your name
- Hosting is in your name
- WordPress is open source (free to use)
- Theme is yours to keep
- No ongoing fees to me (unless support contract)

**"What happens if our web person leaves?"**
- Documentation stays with the site
- Any WordPress developer can help
- Theme is standard WordPress code
- No proprietary lock-in

**"Should we accept donations online?"**
- Once 501(c)3 is approved: Yes
- Plugin recommendation: GiveWP (see PLUGINS-RECOMMENDED.md)
- Can add later when ready
- Not complicated to set up
