# Recommended Plugins for Rivers Area Memory Café

This document lists essential and recommended plugins to enhance your WordPress site. Install only what you need to keep your site fast and secure.

---

## ESSENTIAL PLUGINS (Highly Recommended)

### 1. Events Calendar Plugin
**Choose ONE of the following:**

#### Option A: The Events Calendar (FREE - RECOMMENDED)
- **Plugin Name**: The Events Calendar
- **Developer**: The Events Calendar Team
- **Cost**: Free (Pro version available)
- **Why**: Industry standard, easy to use, perfect for recurring events
- **Installation**: Plugins > Add New > Search "The Events Calendar"

**Setup for RAM Café:**
1. Install and activate the plugin
2. Go to Events > Settings
3. Set default venue to "Fergus Falls YMCA"
4. Create a recurring event:
   - Event title: "Monthly Memory Café Meeting"
   - Recurrence: Monthly on third Tuesday
   - Time: 1:00 PM - 3:00 PM
   - Venue: Fergus Falls YMCA

**Features:**
- Easy recurring event setup (perfect for third Tuesday meetings)
- Calendar and list views
- Venue management
- Google Calendar integration
- Works seamlessly with your theme

#### Option B: Events Manager (FREE Alternative)
- **Plugin Name**: Events Manager
- **Developer**: Marcus Sykes
- **Cost**: Free (Pro available)
- **Why**: Powerful features, good recurring event support
- **Installation**: Plugins > Add New > Search "Events Manager"

## PLUGINS TO AVOID

### Don't Install:
- **Page Builders** (Elementor, Divi, etc.) - Your theme is already well-designed
- **Multiple SEO plugins** - Choose ONE SEO plugin only
- **Too many social sharing plugins** - Keep it simple
- **Plugins you don't actively use** - They slow down your site

## Plugin Installation Steps

1. Log into WordPress admin
2. Go to **Plugins > Add New**
3. Search for the plugin name
4. Click **Install Now**
5. Click **Activate**
6. Follow the setup wizard if one appears

---

## Plugin Maintenance

### Keep Plugins Updated
1. Go to **Dashboard > Updates**
2. Check the box next to plugins with updates
3. Click **Update Plugins**

**IMPORTANT:** Always backup before updating plugins!

### Deactivate Unused Plugins
1. Go to **Plugins > Installed Plugins**
2. Find plugins you're not using
3. Click **Deactivate**
4. Then click **Delete**

## Events Calendar Setup Guide

### Setting Up Your Monthly Meeting

1. **Install The Events Calendar**
2. Go to **Events > Add New**
3. Fill in the details:

**Event Title:** Monthly Memory Café Meeting

**Event Description:**
```
Join us for our monthly gathering at the Fergus Falls YMCA. This is a welcoming space for individuals with memory loss and their care partners to socialize, participate in activities, and connect with others in the community.

Activities include:
- Social time and refreshments
- Memory-friendly activities
- Caregiver support
- Community connections

No registration required. Free and open to all.
```

**Date & Time:**
- Start Date: [Next third Tuesday of the month]
- Start Time: 1:00 PM
- End Time: 3:00 PM
- **Check the box: "This is a recurring event"**
- Recurrence: "Monthly"
- Recurs on: "Third Tuesday"

**Venue:**
- Venue Name: Fergus Falls YMCA
- Address: [Add the full YMCA address]

4. Add a featured image (photo from a past event if available)
5. Click **Publish**

### Adding Special Events

For workshops or special activities:
1. Go to **Events > Add New**
2. Create a single event (don't check recurring)
3. Set specific date, time, and location
4. Add any registration requirements in the description
5. Click **Publish**

## Contact Form Setup Guide

### Basic Contact Form with Contact Form 7

1. **Install Contact Form 7**
2. Go to **Contact > Contact Forms**
3. You'll see a default form - click to edit it

**Default form code (feel free to use as-is):**
```
<label> Your name
    [text* your-name autocomplete:name] </label>

<label> Your email
    [email* your-email autocomplete:email] </label>

<label> Subject
    [text* your-subject] </label>

<label> Your message
    [textarea your-message] </label>

[submit "Send"]
```

4. Go to the **Mail** tab
5. Make sure "To:" is set to your organization's email
6. Save the form

7. Copy the shortcode (looks like `[contact-form-7 id="123" title="Contact form 1"]`)

8. Go to **Pages** and edit your Contact page
9. Add a **Shortcode block** (click the + button and search "shortcode")
10. Paste the shortcode
11. Click **Update**

Your contact form is now live!
